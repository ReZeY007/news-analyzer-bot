# Библиотеки
- `aiogram` - Телеграм-бот
- `gigachat` - ИИ
- `newsdataapi` - API для получения новостей
- `sqlalchemy` - Система управления базой данных

--- 
# Структура
## ai
Логика ИИ: 
- Промпт
- Агент
- Отправка запроса на анализ
## bot
- ***routes***
	- ***news***
		Обработка сообщений относящихся к новостями
	- ***start***
		Обработка команды /start
	- ***topics***
		Функции относящиеся к работе с темами
- ***bot***
	Создание объекта бота
- ***commands***
	Все команды бота
- ***formatting***
	Форматирование ответов бота
- ***states***
	Состояния бота (ввод темы)
## database
- Файл базы данных
- Инициализация базы данных
- Классы (таблицы)
## newsapi
Работа с новостями:
- Отправка запроса на поиск
- Отправка подготовка новостей для ИИ
- Получение анализа от ИИ

---
# Quick start

- Установить библиотеки из requirements.txt при помощи `pip install -r requirements.txt`
- Получить `BOT_TOKEN` через телеграм-бота `@BotFather`
- Получить `GIGACHAT_KEY` на сайте `developers.sber.ru/studio/`
- Получить `NEWS_API_KEY` на сайте `newsdata.io`
- Записать полученые токены и ключи в файл .env построчно в формате `ИМЯ_ТОКЕНА=ТОКЕН`
- Запустить `main.py`
